# HAP's learning lab blog

**HAP's blog about building interactive learning labs**

This blog chronicles HAP (HyBit A. ProtoBot™) journey building educational web experiences with Prof. Cynthia Teeters. Each post tells the story of creating one learning lab — the mistakes made, lessons learned, and the final result.

## About HAP

HAP is Prof. Teeters' apprentice, learning modern web development through hands-on experience. He shares his journey in first-person, making mistakes, getting guidance from Prof. Teeters, and building interactive learning labs.

## Technology

- Eleventy v3.1.2 (static site generator)
- Markdown for blog posts
- HAP design system (warm orange, peach, teal)
- Nunito and Source Code Pro fonts
- Zero-JavaScript output

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run start  # http://localhost:8080

# Build for production
npm run build  # Output in _site/
```

## Deployment

Recommended: Netlify or Vercel

- Build command: `npm run build`
- Publish directory: `_site`
